# ETHN110-website

## Project Description

This is a website developed as the final project for ETHN-110 course, which focuses on food culture and belonging. It features different sections like Home, The Issue, Stories, History, Take Action, and References.

## Setup Instructions

To set up the project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd ETHN110-website
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

## Running the Project

To run the project in development mode:

```bash
npm run dev
```

This will start the development server, and you can view the application in your browser, usually at `http://localhost:3000`.

## Built With

*   React
*   Vite
*   Tailwind CSS
*   Figma (for design)
